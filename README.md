# Iris Species Classifier App Santiago Diaz y Samuel COrrea
Aplicación local desarrollada en Python siguiendo arquitectura modular.

### Requisitos:
- Entorno Virtual: `env_iris`
- Modelo entrenado: `modelo_iris.h5`

### Ejecución:
1. Activar entorno: `env_iris\Scripts\activate`
2. Correr app: `python main.py`
